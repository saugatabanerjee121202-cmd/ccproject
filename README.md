# Customer Churn Prediction Using Machine Learning

## Overview

This project predicts whether a customer is likely to churn using a Logistic Regression model trained on the IBM Telco Customer Churn Dataset.

## Features

* Data preprocessing and cleaning
* Feature encoding
* Logistic Regression model
* Churn probability prediction
* Interactive Streamlit web application

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* Joblib

## Dataset

IBM Telco Customer Churn Dataset

## Model Performance

* Accuracy: ~78%

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run streamlit_app.py
